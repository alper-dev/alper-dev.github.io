# alperdev.codes

Personal portfolio built with Astro, React, Tailwind v4, and Framer Motion.

## Development

```bash
bun install
bun run dev
```

## Content

- `src/content/site.json` - bio, socials, labels
- `src/content/projects.json` - project list

## Deploy

Push to `main`. GitHub Actions builds and deploys to GitHub Pages.
